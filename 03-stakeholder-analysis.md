# Stakeholder Analysis

* [cite_start]**Customer Support Agents:** Require the system to allow instant retrieval of "OPEN" tickets for a specific user to resolve issues efficiently. [cite: 16]
* [cite_start]**Business Analysts:** Utilize the analytics view to enable platform-wide analytics across specific event types without scanning individual user profiles. [cite: 19]
* [cite_start]**AI & Data Science Team:** Manage the research and algorithmic design phase to classify sentiment and predict churn. [cite: 28, 29]
* [cite_start]**System Administrators:** Responsible for the Docker-deployed backend and managing local server nodes equipped with high-parameter hardware (e.g., RTX 3090/4090 GPU arrays). [cite: 10, 30]
